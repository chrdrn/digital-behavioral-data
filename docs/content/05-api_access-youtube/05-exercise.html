Showcase: YouTube

Focus on YouTube Data Tools

Background

Practical application of the YouTube Data Tool (YTDT) using the example of Mai Thi Nguyen-Kim ( | ) and her -Channel maiLab.

Assignments

  1. Use the Channel Search Function of the YTDT to find the correct channel ID for -Channel maiLab. Therefore, enter “mailab” in the field search query and download the results as .csv. Open the file and search extract the correct channel ID.
    If in doubt, use Channel Info function to check if the selected ID matches the channel description.
